# A-Watch

Приложение для просмотра аниме на Flutter.

## Архитектура

- Clean Architecture
- BLoC для управления состоянием
- MediaKit для видео-плеера
- Hive для кэширования

## Функции

- Каталог аниме
- Просмотр видео
- Коллекции
- Топ-100
- Фильмы и сериалы

## Запуск

1. Установить Flutter
2. `flutter pub get`
3. `flutter run`

## Сборка

- Android: `flutter build apk`
- iOS: `flutter build ios`
- Windows: `flutter build windows`
